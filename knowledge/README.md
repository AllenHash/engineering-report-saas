# 知识库 - 双重分类系统

## 支持两种查找方式 ✅

---

## 方式一：按来源分类

```
knowledge/
└─ 按来源分类/
    ├── 公众号/       （2篇）
    │   └── OpenClaw相关文章
    │
    └── B站视频/      （4个）
        └── OpenClaw相关视频
```

---

## 方式二：按主题分类（同一批内容）

```
knowledge/
└─ OpenClaw/          （6个）
    ├── posts/          （2篇）
    └── videos/         （4个）

├── AI_Agents/         （待添加）
├── 工程报告/           （待添加）
├── 运营/              （待添加）
└── 每日总结/          （待添加）
```

---

|------|------|

## 使用方法

- **按来源找**：打开"按来源分类"文件夹
- **按主题找**：打开对应主题文件夹
- 两种方式内容相同，互为备份
