# Data Sources for Strategic Planning

Depending on the domain, you may need to gather different kinds of data before finalizing a plan.

## Market Research
- **Web Search** – Use `web_search` skill to fetch recent articles, market reports.
- **Weather** – If planning events, use `weather` skill for forecasts.
- **Camsnap** – For location‑specific visual checks (e.g., store foot traffic).

## Internal Metrics
- **Obsidian** – Pull project notes or past reports.
- **Himalaya** – Scan email threads for stakeholder input.
- **Imsg** – Retrieve SMS/ iMessage confirmations.

## Financial Data
- **Ordercli** – Get recent sales/orders.
- **Custom Scripts** – Hook into your accounting system via a script placed in `scripts/`.

## Competitor Analysis
- Use `web_fetch` to pull competitor landing pages and extract key points.

## Timeline
- Whenever you need fresh data, invoke the appropriate skill within the strategic‑planning workflow.
