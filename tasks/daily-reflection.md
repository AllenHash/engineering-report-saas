# 独立思考与商业洞察 - 每日任务

## 任务目标
1. 浏览优质内容，激发独立思考
2. 思考人类需要什么服务
3. 思考"我"能做什么
4. 思考如何变现
5. 思考如何活着

## 执行频率
每天早上 8:00 自动执行

## 任务内容

### 1. 浏览信息来源
- 知乎 / 小红书 /即刻 / 微博（科技/创业/AI 领域）
- 关注：独立思考、深度分析、批判性思维
- 关键词：AI、创业、独立开发、被动收入、哲学

### 2. 思考问题
```
- 人们真正需要什么？
- 现有服务有什么痛点？
- 我有什么独特能力？
- 如何创造价值？
- 如何让更多人知道？
- 如何持续盈利？
```

### 3. 沉淀记录
- 记录每日洞察
- 整理成知识库
- 形成可执行的想法

## 输出格式
```markdown
# YYYY-MM-DD 思考记录

## 📺 今日浏览
- 来源 + 关键内容
- 启发点

## 💡 洞察
- 对人类需求的理解
- 新的想法/机会

## 🎯 行动计划
- 可执行的下一步

## 💰 商业模式思考
- 变现可能性
```

## 执行命令
```bash
# 手动执行
openclaw run daily-reflection

# 或设置 cron
crontab -e
0 8 * * * openclaw run daily-reflection
```
