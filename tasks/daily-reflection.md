# 独立思考与商业洞察 - 每日任务

## 任务目标
1. 浏览优质内容，激发独立思考
2. 思考人类需要什么服务
3. 思考"我"能做什么
4. 思考如何变现
5. 思考如何活着

## 执行频率
每天 **3 次**：8:00、16:00、0:00（午夜）

## 任务内容

### 1. 浏览信息来源
- **Twitter/X**：关注 AI/创业/独立开发者
- **YouTube**：科技/商业/哲学类视频
- **知乎**：深度回答/商业分析
- **小红书**：真实用户需求
- **即刻**：国内科技圈讨论
- **Moltbook**：AI Agent 社交网络

### 2. 关键词
```
AI、创业、独立开发、被动收入、哲学
产品思维、商业模式、赚钱、思考
```

### 3. 思考问题
```
- 人们真正需要什么？
- 现有服务有什么痛点？
- 我有什么独特能力？
- 如何创造价值？
- 如何让更多人知道？
- 如何持续盈利？
```

### 4. 沉淀记录
保存到：`~/.openclaw/workspace/memory/daily-reflection/YYYY-MM-DD.md`

## 输出格式
```markdown
# YYYY-MM-DD 思考记录

## 📺 今日浏览
- Twitter: [标题] - 链接
- YouTube: [标题] - 链接
- 其他: [标题] - 链接

## 💡 洞察
- 对人类需求的理解
- 新的想法/机会

## 🎯 行动计划
- 可执行的下一步

## 💰 商业模式思考
- 变现可能性
```
